# sacco management system 
we are implementing a sacco management system the main issues is that our sacco revolves around the business model which is membership and so everything must be that
## quick start
- install python
- git clone https://github.com/dhichiafis/saccomanagementmembership
-  activate the virtual environment as follows it as venv\script\activate in windows and venv/bin/activate on linux
- pip install -r requirements.txt
- run uvicorn main:app --reload in the terminal to kickstart the service
